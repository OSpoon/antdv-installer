# antdv-installer

一键将 Ant Design of Vue 组件库安装在使用 Vite 构建的 Vue-ts 项目

## 使用方式

使用 npx 运行:

```shell
npx antdv-installer
```

全局安装后运行:

```shell
npm i -g -D antdv-installer

```

```shell
antdv

or

antdv-installer
```

项目安装后运行:

```shell
npm i -D antdv-installer

```

```shell
npm run antdv

or

npm run antdv-installer
```
